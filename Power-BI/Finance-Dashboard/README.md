# 💰 Finance Dashboard

## 📌 Project Overview

This Power BI dashboard provides financial insights by analyzing sales, profit, gross sales, units sold, and country-wise performance.

---

## 🛠 Tools Used

- Power BI
- Excel Dataset

---

## 📊 KPIs

- Total Sales
- Total Profit
- Total Units Sold
- Total Gross Sales

---

## 📈 Visualizations

- Sales by Country
- Profit by Segment
- Monthly Sales Trend
- Sales by Product
- Sales by Discount Band
- Profit by Country

## 🎛 Filters

- Segment
