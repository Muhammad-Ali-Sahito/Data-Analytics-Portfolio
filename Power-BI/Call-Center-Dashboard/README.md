# 📞 Call Center Dashboard

## Project Overview
This Power BI dashboard analyzes call center performance using KPI cards, interactive charts, and dynamic slicers.

## Dashboard Highlights
- Total Calls
- Total Revenue
- Resolved Calls
- Resolution Rate
- Average Call Duration
- Average Customer Rating

## Tools Used
- Power BI
- DAX

## Dashboard Features
- KPI Cards
- Interactive Charts
- Dynamic Slicers
