# 📦 Inventory Dashboard

## 📌 Project Overview

This Power BI dashboard provides an overview of product inventory, sales, orders, and available stock. It helps analyze inventory performance and product sales.

---

## 🛠 Tools Used

- Power BI
- CSV Dataset

---

## 📊 KPIs

- Total Products
- Total Sales
- Total Orders
- Available Stock

---

## 📈 Visualizations

- Sales by Product (Clustered Bar Chart)
- Sales by Category (Clustered Column Chart)
- Orders by Category (Donut Chart)
- Top Products by Sales (Bar Chart)

---

## 🎛 Filters

- Category Slicer
- Product Slicer
